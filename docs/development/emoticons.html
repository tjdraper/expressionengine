Emoticon Development¶

Modifying or creating new sets of Emoticons for use with ExpressionEngine is fairly straightforward. Glyph translations are contained in the /system/ee/legacy/modules/emoticon/emoticons.php file. This file contains an array, $smileys that provides instructions as to what glyphs should be replaced by which images, and what the properties of the image files are. The files themselves can be replaced or modified so that completely different Emoticon packs can be created and used.

smiley¶

The glyph / text you wish to be replaced by a particular smiley

image name¶

The file name for the smiley image

width¶

The width, in pixels, of the smiley image

height¶

The height, in pixels, of the smiley image

alt¶

The text that will be used for the image alt="" attribute

Emoticon Images¶

The file names in the $smileys array correspond to the files in the directory specified as the “smilies” directory in the Control Panel at Settings ‣ Content & Design.
